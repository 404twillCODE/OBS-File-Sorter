What do these folders mean?

Source folder — Your main OBS folder where recordings and clips are stored. This is where the program looks for files to sort.

Backtrack folder — For clips from the Aitum Vertical Discord plugin. Sorted files that contain "backtrack" in the name go here.

Replay folder — Clips you get from using OBS to clip (replay buffer). Files with "replay" in the name go here.

Recording folder — Your main recordings. Everything else (that isn’t backtrack or replay) goes here, organized by date.

Vault destination — A place to save/copy files. The program creates a vault folder in each date folder named after the vault destination (for example, if your vault folder is called "Keepers", each date folder gets a "Keepers" folder). Any subfolders inside your vault destination are mirrored there, but no clips are created automatically.

By default the program only moves or copies files. In Settings you can optionally enable auto-deletion of short clips or old folders.
